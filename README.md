# EnglishMemoApp
英会話で新しくフレーズを覚えたい時に、シチュエーション別にメモを残せるアプリです。

# Description
フロントエンドはReact Hooks
バックエンドはDjango rest framework
を使いました。
追加したフレーズは編集削除が可能です。

# Usage
## trains
もう一つのリポジトリEnglishMemoApp_APIがアプリのAPIになります。

# Requirements
- Python 3.7.3
- conda 4.7.12
- node_modules
- django
